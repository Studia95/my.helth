# my.helth

PWA-приложение для контроля приёма лекарств.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## GitHub Pages

```bash
npm run build
```

Затем можно залить папку `dist` в GitHub Pages или настроить GitHub Actions для публикации собранного приложения.

## Возможности

- Лекарства
- Чек-лист
- Календарь
- Статистика
- Уведомления
- Экспорт/импорт
- Оффлайн-режим

## Ограничения уведомлений

Локальные уведомления в браузерах могут работать по-разному. На iPhone и Android для стабильной работы лучше установить приложение на главный экран. Без backend точные push-уведомления при полностью закрытом браузере не гарантируются.
