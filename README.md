# my.helth

PWA-приложение для контроля приёма лекарств.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## GitHub Pages

Публикация настроена через GitHub Actions. После push в `main` workflow собирает приложение в `dist` и публикует готовую сборку на GitHub Pages.

В настройках репозитория GitHub Pages должен быть выбран источник `GitHub Actions`.

## Возможности

- Лекарства
- Чек-лист
- Календарь
- Статистика
- Уведомления
- Экспорт/импорт
- Оффлайн-режим

## Ограничения уведомлений

Локальные уведомления в браузерах могут работать по-разному. На iPhone и Android для стабильной работы лучше установить приложение на главный экран. Без backend точные push-уведомления при полностью закрытом браузере не гарантируются.
